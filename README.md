## Installing

Remove the current neovim configuration and existing data:

```bash
# Linux / MacOS (unix)
rm -rf ~/.config/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.local/share/nvim
```

Then clone this repo:

```bash
git clone https://github.com/angelsolaorbaiceta/nvchadconfig ~/.config/nvim
```
